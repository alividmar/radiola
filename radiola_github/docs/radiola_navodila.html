Začetek uporabe

Zaženi program: python3 radiola.py (potrebuješ Python 3.8+ in pakete: pip install pillow requests mutagen)
Klikni gumb ▶ Predvajaj ali klikni na ikono postaje da začneš predvajanje.

Upravljanje postaj

✏️
Uredi postaje
Klikni gumb ✏ Uredi postaje. Dodaj, uredi ali izbriši postaje. Vsaka postaja ima: ime, URL streama, ikono in opcijsko ključno besedo za dvig glasnosti.
📋
Uvoz / Izvoz
V urejevalniku sta gumba 📂 Uvozi .txt in 💾 Izvozi .txt. Format: Ime = URL = base64_ikona (ikona je neobvezna).
🖼️
Ikone postaj
Klikni 📁 v urejevalniku da odpreš brskalnik ikon s predogledom. Ikone se shranjujejo v visoki ločljivosti (225×225 px). Deluje s PNG, JPG, WEBP.
🌐
Samodejni prenos ikon
Gumb 🌐 Ikona poskuša sam prenesti logo postaje iz interneta (Google Favicon, HTML scraping). Deluje za večino znanih postaj.
↕️
Razporeditev
Postaje v urejevalniku premakneš z gumboma ▲ ▼. Prvih 16 postaj (2 vrsti po 8) je vidnih kot ikone v glavnem oknu.
🔍
Iskanje in filtriranje
V urejevalniku je iskalno polje na vrhu. Piši ime postaje in seznam se sproti filtrira. Gumb Podvojene poišče morebitne duplikate.

Glasnost in dvig pri govoru

🔊
Drsnik glasnosti
Drsnik 🔊 v zgornjem delu neposredno nastavi glasnost predvajanja (0–100%). Nastavitev se shrani in obnovi ob naslednjem zagonu.
🎙️
Samodejni dvig pri govoru
Ko program zazna govorno vsebino (po naslovu pesmi/oddaje), samodejno dvigne glasnost. Dvig nastavi z drsnikom +0%–+60% ali ga izklopi z dropdownom.
Primer: Nastavi ključno besedo glasovanje;govorna;mix — ko naslov pesmi vsebuje katero koli od teh besed (delno ujemanje, ne glede na vejice), program dvigne glasnost.

Nastavitev ključnih besed

Odpri Nastavitve → Ključna beseda…
Vpiši eno ali več besed, ločenih s podpičjem ; npr. govorna;glasovanje;pogovor
Program išče podbesedilo — glasov zazna tudi glasovanje, glasovanju, glasoval.
Za posamezno postajo nastavi svojo ključno besedo v urejevalniku (polje Ključna beseda postaje); prazno pomeni globalna nastavitev.

Timer za izklop

Nastavi čas v minutah (sprtni gumb) in pritisni ▶ ob timerju. Program bo postopoma utišal glasnost v zadnjih 60 sekundah in nato ustavil predvajanje. Izklopi z ×.

Nasvet za spanje: Odkljukaj "Utišaj pred izklopom" če ne želiš postopnega utišanja — radio se bo ustavil nenadoma.

🌍 Iskanje postaj

Dialog za iskanje postaj ima dva zavihka: Radio Browser (brez registracije) in RapidAPI (z brezplačnim ključem).

🆓
Radio Browser — brez registracije
~35.000 aktivnih postaj, brez omejitev zahtev, odprtokoden, preverja delovanje streamov vsak dan.
🔑
RapidAPI — z brezplačnim ključem
90.000+ postaj iz 200+ držav. 500 iskanj/mesec brezplačno. Boljše ikone in metapodatki.

🆓 Radio Browser (priporočeno za začetek)

Klikni ✏ Uredi postaje → 🌍 Poišči postaje.
Zavihek 🆓 Radio Browser je že aktiven — ni potrebna nobena nastavitev.
Izberi državo, žanr, razvrsti po glasovih/priljubljenosti/imenu/bitratu.
Obkljukaj "Samo aktivne" da vidiš le postaje s delovnim streamom.
Klikni 🔍 Poišči → izberi postaje → ✓ Dodaj izbrane.

🔑 RapidAPI (več postaj)

Registracija na rapidapi.com (brezplačno) → poišči "50K Radio Stations" → Subscribe.
V Radioli: Nastavitve → RapidAPI ključ… → prilepi ključ → Shrani.
V iskanju preklopni na zavihek 🔑 RapidAPI in išči kot pri Radio Browser.
Nasvet: Začni z Radio Browser — brez registracije, brez omejitev. Dodaj RapidAPI ključ samo če potrebuješ dostop do postaj ki jih Radio Browser nima.

Skupne funkcije iskanja

Ctrl+klik za izbor več postaj, Ctrl+A za vse. Dvojni klik prikaže detajle postaje. Ikone se prenesejo samodejno ob dodajanju.

Getting Started

Launch the app: python3 radiola.py (requires Python 3.8+ and packages: pip install pillow requests mutagen)
Click ▶ Play or click a station icon to start streaming.
Adjust volume with the 🔊 slider at the top of the window (0–100%).
Switch stations by clicking an icon, selecting from the dropdown, or using the ◀ ▶ buttons.

Managing Stations

✏️
Edit Stations
Click ✏ Edit stations. Add, edit or delete stations. Each station has: name, stream URL, icon, and an optional keyword for the volume boost feature.
📋
Import / Export
Use 📂 Import .txt and 💾 Export .txt in the editor. Format: Name = URL = base64_icon (icon is optional).
🖼️
Station Icons
Click 📁 in the editor to open the icon browser with preview. Icons are stored at full resolution (225×225 px). Supports PNG, JPG, WEBP.
🌐
Automatic Icon Fetch
The 🌐 Icon button automatically downloads the station logo from the internet (Google Favicon, HTML scraping). Works for most known stations.
↕️
Reordering
Use ▲ ▼ buttons in the editor to reorder stations. The first 16 stations (2 rows of 8) appear as icon buttons in the main window.
🔍
Search & Filter
Type in the search box at the top of the editor to filter stations in real time. The Duplicates button finds any duplicate entries.

Volume & Speech Boost

🔊
Volume Slider
The 🔊 slider at the top directly controls playback volume (0–100%). The setting is saved and restored on next launch.
🎙️
Automatic Speech Boost
When the app detects spoken content (by matching the now-playing title), it automatically raises the volume. Set the boost amount (+0–+60%) with the slider, or disable via the dropdown.
Example: Set the keyword to vote;talk;speech;mix — whenever the now-playing title contains any of these words (partial match, case-insensitive), the volume will be raised.

Setting Up Keywords

Go to Settings → Keyword…
Enter one or more words separated by semicolons ; e.g. talk;news;vote;speech
The app does partial (substring) matching — vote will also match voter, voting, voted.
Set a per-station keyword in the editor (Station keyword field); leave blank to use the global setting.

Sleep Timer

Set the number of minutes using the spin box, then press ▶ next to the timer. The app will gradually fade the volume during the last 60 seconds and then stop. Cancel with ×.

Tip: Uncheck "Fade before stop" if you don't want the gradual fade — the stream will stop immediately when the timer expires.

Icon Browser

Clicking 📁 in the station editor opens a built-in image browser with live preview:

On first launch it opens in the program folder (where radiola.py is located).
Navigate with ⬆ Up and 🏠 Home, or click a folder thumbnail in the grid.
Click an image thumbnail — a 140×140 px preview appears on the right.
Confirm with ✓ Confirm. The icon is saved at full resolution (225×225 px).
Next time, the browser remembers the last folder you used.

🌍 Station Search

The search dialog has two tabs: Radio Browser (no registration) and RapidAPI (free key required).

🆓
Radio Browser — no registration
~35,000 active stations, unlimited requests, open source, stream health checked daily.
🔑
RapidAPI — free key required
90,000+ stations from 200+ countries. 500 searches/month free. Better icons and metadata.

🆓 Radio Browser (recommended to start)

Click ✏ Edit stations → 🌍 Find stations.
The 🆓 Radio Browser tab is active by default — no setup needed.
Select country, genre, and sort order (votes / popularity / name / bitrate).
Check "Active only" to show only stations with working streams.
Click 🔍 Search → select stations → ✓ Add selected.

🔑 RapidAPI (more stations)

Sign up at rapidapi.com (free) → find "50K Radio Stations" → Subscribe.
In Radiola: Settings → RapidAPI key… → paste key → Save.
Switch to the 🔑 RapidAPI tab in the search dialog and search as usual.
Tip: Start with Radio Browser — no registration, no limits. Add a RapidAPI key only if you need stations not found in Radio Browser.

Common search features

Ctrl+click to select multiple stations, Ctrl+A for all. Double-click shows station details. Icons are downloaded automatically when adding.

Troubleshooting

ProblemSolution
Radio won't playMake sure mpv is installed. Check the player status in the window header.
Icon not showing (circle with initials)The icon may be stored at too low a resolution. Click 🌐 in the editor to re-download it.
Volume boost not workingCheck that the dropdown is not set to "Off" and that keywords are correctly configured.
Empty station buttonsStations 1–16 appear as icons. Add stations in the editor or move existing ones to the top 16 positions.
Settings not savingMake sure Python has write permission in the program folder (settings.json, stations.json).
